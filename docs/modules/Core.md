# Module: Core

Iniquity Core

**`summary`** The super cool command line interface to Iniquity.

**`example`** Invoking via the shell
```shell
iq cli -h
```

**`example`** Invoking via yargs programatically
```typescript
import CLI from "@iniquitybbs/cli"
const cli: yargs.CommandModule = new CLI()
```
